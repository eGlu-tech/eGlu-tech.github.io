# eGlu.tech

**Built with simplicity. Minimalist Engineering.**

eGlu.tech, a fast blog built with Astro, prioritizing the "Vanilla Web" (Standard Web APIs and Vanilla CSS) over heavy frameworks. It serves as a practical exploration of clean, maintainable architecture and intentional design.

## 🪐 Philosophy

- **Vanilla Everything:** A commitment to standard Web APIs. No Tailwind, no CSS-in-JS, and no unnecessary dependencies.
- **Warm Minimalism:** A human-centric, technical aesthetic built on dark stone tones (`#1c1917`) and amber accents (`#d97706`).
- **Technical Precision:** Typography chosen for legibility and structure, using `Space Grotesk` (headings) and `Inter` (body).
