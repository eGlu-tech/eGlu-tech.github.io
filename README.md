# eGlu.tech

**Minimalist Engineering & The Vanilla Web.**

eGlu.tech is a high-performance blog built with Astro, prioritizing the "Vanilla Web" (Standard Web APIs and Vanilla CSS) over heavy framework abstractions. It serves as a practical exploration of clean, maintainable architecture and intentional design.

## 🪐 Philosophy

- **Zero-JS by Default:** Utilizing Astro's Island Architecture to ship minimal JavaScript. Interactive components are the exception, not the rule.
- **Vanilla Everything:** A commitment to standard Web APIs. No Tailwind, no CSS-in-JS, and no unnecessary dependencies.
- **Warm Minimalism:** A human-centric, technical aesthetic built on dark stone tones (`#1c1917`) and amber accents (`#d97706`).
- **Technical Precision:** Typography chosen for legibility and structure, using `Space Grotesk` (headings) and `Inter` (body).

## 🛠 Foundation

- **Framework:** [Astro](https://astro.build) (Static Site Generation).
- **Styling:** Vanilla CSS with CSS Variables.
- **Content:** Markdown-driven via Astro Content Collections.
- **Deployment:** Automated via GitHub Actions to [eglu.tech](https://eglu.tech).
