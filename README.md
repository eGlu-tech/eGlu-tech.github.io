# eGlu.tech

**Minimalist Engineering & The Vanilla Web.**

eGlu.tech is a manifesto in motion—a high-performance, minimalist blog built to prove that the modern web doesn't need to be heavy. It is a sanctuary for clean code, intentional design, and the "Vanilla Web."

## 🪐 The Vision

We believe in building for the next decade, not the next trend. This repository isn't just a site; it's a commitment to three core pillars:

### 1. Zero-JS by Default
The front-end shouldn't be a burden. By utilizing **Astro's Island Architecture**, we ship zero JavaScript to the client by default. Interactive components are treated as exceptions, not the rule.
- **Vanilla Everything:** Standard Web APIs over heavy framework abstractions.
- **Performance First:** Every byte is audited for necessity and impact.

### 2. Warm Minimalism
We reject the sterile, high-contrast aesthetics of the modern tech industry. eGlu.tech is built on a "Warm Minimalist" design system:
- **Aesthetic:** Dark stone tones (`#1c1917`) paired with amber accents (`#d97706`).
- **Typography:** Technical precision meeting human approachability via `Space Grotesk` (headings) and `Inter` (body).
- **Breathable Layouts:** Using negative space to reduce cognitive load and prioritize reading rhythm.

### 3. Engineering for the Long Haul
Simplicity is the ultimate form of sophistication and maintainability.
- **Open Standards:** We prioritize open formats and avoid vendor lock-in.
- **Surgical Updates:** As defined in `GEMINI.md`, our workflow is autonomous and precise, favoring small, meaningful iterations over monolithic changes.

---

## 🛠 Technical Foundation

- **Framework:** [Astro](https://astro.build) (Static Site Generation).
- **Styling:** Vanilla CSS with CSS Variables. No Tailwind, no CSS-in-JS.
- **Content:** Markdown-driven via Astro Content Collections.
- **Deployment:** Automated via GitHub Actions to [eglu.tech](https://eglu.tech).
